# Getting Started with OryonAi

Welcome to OryonAi! This guide will help you get started with our platform.

## Quick Start

1. **Create an Account**: Sign up for a free account to get started
2. **Choose Your Plan**: Select a subscription tier that fits your needs
3. **Access the Dashboard**: Navigate to your dashboard to explore features
4. **Read the Documentation**: Learn about all available features and APIs

## Subscription Tiers

### Free Tier
- Basic dashboard access
- Limited API calls
- Community support

### Lite Tier
- Everything in Free
- Analytics dashboard
- Increased API limits
- Email support

### Pro Tier
- Everything in Lite
- Advanced analytics
- Custom integrations
- Priority support

### Advanced Tier
- Everything in Pro
- Dedicated support
- Custom SLAs
- Unlimited API calls
